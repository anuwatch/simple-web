WELCOME TO HPE LAB

This is a demonstration for end to end CI/CD. You will see build, provisioning, test and many more. This lab will show you how can we provision from Infrastructure level to application level.

@Warot Asawa  - As of 26th JUNE 2020

Fully Automated

Infrastructure

With HPE Nimble Storage and Ansinimble roles. You can simple provision volume from nimble array and mount to your host easily.

Environment

With automation, you can setup Python , Apache server, or any other environment for your apps in few seconds.

Application

Application deployment has never been easier when you can pull new code from Git server and run the app in one steps.

Full Stack deployment

  • 1. Create Volume on Nimble Array
  • 2. Automatic mount and create volume on hosts
  • 3. Setup environment by install all required packages
  • 4. Deply application from GitHub
